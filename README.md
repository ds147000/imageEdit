# iamgeEdit
一个轻量级到js图片编辑插件，不依赖任何第三方，可单独和兼容第三方框架使用



###快速使用

------------


##### 	在浏览器中使用
```js
<script src="/dist/image-edit.min.js"></script>
<script>
	var app = new ImageEdit('body')
</script>
```

##### 在webpack中使用
```js
npm i image-edit
```
```js
	import ImageEdit from 'image-edit'
	
	new ImageEdit('body')
```
